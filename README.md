# vim4roxygen2
##usage

1. add the vimrc content to your .vimrc file
2. put your cursor on the function name , press F3 to generate the roxygen template

following is an example:

```
#’ title
#’
#’ description
#’
#’ @param a value
#’ @param b value
#’ @param d value
#’ @param e value
#’ @return returndes
#’ @export
#’ @examples
#’ x=c(1,2,3)
afunction<-function(a,b=”cc”,d=1,e){
return 1
}
```
